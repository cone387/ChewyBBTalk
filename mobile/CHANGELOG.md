# ChewyBBTalk Mobile 更新日志

## v1.1.0 (2026-04-17)

### 新功能

**分享功能**
- 支持将碎碎念分享到微信等应用（系统分享面板）
- 自动组装正文 + 标签为分享文本
- 图片附件随分享内容一并发送
- Web 端支持 Web Share API，不可用时降级为剪贴板复制
- 分享失败/取消时提供"复制到剪贴板"降级选项

**手势操作**
- 左滑碎碎念卡片显示删除操作区域
- 右滑碎碎念卡片显示置顶/取消置顶操作区域
- 60fps 原生动画驱动，弹性回弹效果
- 同一时刻只有一张卡片展开

**Markdown 编辑实时预览**
- 编辑页新增编辑/预览模式切换按钮
- 预览模式使用与首页卡片一致的 Markdown 渲染样式
- 切换回编辑模式时保持内容和光标位置不变
- 空内容时显示"暂无内容可预览"占位提示

**批量操作**
- 长按碎碎念卡片 500ms 进入批量选择模式
- 支持全选、批量删除、批量改标签、批量改可见性
- 操作过程中显示进度指示器，禁用重复提交
- 部分失败时汇总显示错误信息
- Android 返回键退出批量模式

**离线缓存**
- API 数据自动缓存到本地 SQLite 数据库
- 离线时从缓存加载数据，显示"离线模式"提示条
- 在线时采用 cache-first + 后台刷新策略
- 离线状态下写操作（新建/编辑/删除）显示友好提示
- 数据管理页面支持清除离线缓存
- 提示条显示"最后同步于 XX 分钟前"

### UI 优化

**P0 — 深色模式适配**
- ComposeScreen 全面适配主题色（40+ 处硬编码颜色替换）
- LoginScreen 全面适配主题色（30+ 处硬编码颜色替换）
- 深色模式下 borderLight 和 surfaceSecondary 颜色区分度优化

**P1 — 交互体验**
- BBTalkCard 残留硬编码颜色替换为主题色
- 可见性切换按钮触摸区域从 23pt 增大到 35pt
- 更多操作按钮 padding 从 2px 增大到 6px
- TagTabs 标签触摸区域增大到 44pt
- UndoToast 位置上移，避免与 FAB 按钮重叠
- OfflineBanner 添加圆角，与卡片风格统一
- BatchToolbar 底部间距优化

**P2 — 设计质量**
- ThemeSettingsScreen 预览边框使用主题色
- SkeletonCard 对屏幕阅读器隐藏
- 关键交互元素添加 accessibilityLabel（FAB、菜单、搜索、更多操作、可见性切换、批量选择）
