# ChewyBBTalk Mobile 产品路线图

## P0 — 上架必须

| 任务 | 说明 | 预估 |
|------|------|------|
| EAS Build 配置 | `eas build --platform ios`，配置证书签名 | 2h |
| Apple Developer 账号 | 注册 + 配置 App ID、Provisioning Profile | 1h |
| Bundle Identifier | `app.json` 设置 `ios.bundleIdentifier` | 5min |
| App 图标 | 设计 1024x1024 图标，配置到 `app.json` | 2h |
| 启动屏 | 设计 splash screen，替换默认 | 1h |
| 隐私政策 | 写隐私政策页面，提供 URL | 1h |
| App Store 截图 | 6.7 寸 + 5.5 寸两套截图 | 1h |
| App Store 描述 | 标题、副标题、关键词、描述 | 30min |
| 公网后端部署 | 部署到云服务器，配置 HTTPS | 2h |

## P2 — 功能增强

| 任务 | 说明 | 预估 |
|------|------|------|
| 后端语音转文字 | Whisper API / faster-whisper，作为设备端 STT fallback | 4h |
| 分享功能 | 分享单条碎碎念到微信/其他 app（文字+图片） | 3h |
| 标签管理 | 独立页面编辑/删除/改颜色/排序 | 3h |
| 搜索历史 | 记录最近搜索词，快速选择 | 1h |
| Markdown 预览 | 编辑页切换预览模式 | 2h |
| 批量操作 | 长按进入多选模式，批量删除/改标签/改可见性 | 4h |
| 存储迁移 | 本地 ↔ S3 附件迁移 | 3h |
